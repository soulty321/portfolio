## 1. 기술적 분석 기초 (5/11-5/14)

### 목표
각 보조지표가 차트에 어떤 시각 요소로 렌더링되는지 설명 가능 수준

### 학습 항목

#### MA (이동평균선)
- [ ] MA5/MA20/MA60/MA120 각각의 의미와 기간별 용도
- [ ] 차트 위 시각 표현: 라인 색상, 두께, 레이블 위치
- [ ] 골든크로스/데드크로스 시각적 패턴
- 학습 메모:

#### BB (볼린저밴드)
- [ ] 상한/중간/하한 3개 라인의 의미
- [ ] 밴드 영역 시각 표현 (fill area, 투명도)
- [ ] MA와의 관계: 중간선 = MA20 (토스증권 chart_ui_analysis.md 결함 1-4 참고)
- 학습 메모:

#### MACD
- [ ] MACD선(본선), 시그널선, 히스토그램 3요소
- [ ] 패널 분리 표현 (메인 차트 아래 별도 패널)
- [ ] 컬러 토큰: 본선 파랑, 시그널 주황 (토스증권 현행)
- 학습 메모:

#### RSI
- [ ] 0-100 범위, 과매수(70+)/과매도(30-) 영역
- [ ] 별도 패널 표현, 영역 fill
- [ ] 컬러 토큰: RSI 자주, 시그널 파랑 (토스증권 현행)

## 3. 토스증권 앱 심층 분석 (5/15-5/21)

### 목표
보조지표 3개 이상 조합 시나리오 스크린샷 10장+, chart_ui_analysis.md 확장

### 분석 시나리오 체크리스트
- [ ] MA(5/20/60) + BB 조합 → 컬러 충돌 관찰
- [ ] MA + MACD + RSI 3패널 → 패널 간 동일 hue 재사용 관찰
- [ ] BB + MACD → BB 중간선 vs MA 색상 불일치 확인
- [ ] 미국 주식(S&P500 ETF) vs 한국 주식 → 글로벌 토큰 동일 여부
- [ ] 멀티차트(2분할, 4분할) → 패널 간 독립 토큰 확인
- [ ] 다크모드 vs 라이트모드 → 컬러 토큰 대비 변화
- [ ] 모바일(MTS) vs PC(WTS) → 동일 지표 표현 차이
