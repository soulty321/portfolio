# 경쟁사 차트 UI 비교 분석 — 토스증권 [Chart]

> 목표: 주요 트레이딩 플랫폼의 차트 UX를 벤치마킹하여, 토스증권만의 차별화된 차트 경험 제안 근거 확보
> JD: "다양한 트레이딩 플랫폼의 차트 UX를 벤치마킹하고, 토스증권만의 차별화된 차트 경험을 제안"

---

## 비교 대상

| 플랫폼 | 유형 | MTS/WTS | 특징 |
|--------|------|---------|------|
| **토스증권** | 국내 (기준) | MTS + WTS | TradingView 기반, 멀티차트 분할 |
| **키움증권 (영웅문)** | 국내 | MTS + WTS(HTS) | 국내 점유율 1위, 전통 HTS |
| **한국투자증권 (eFriend)** | 국내 | MTS + WTS | 종합증권사, 다양한 차트 도구 |
| **Robinhood** | 해외 | MTS + Web | 심플 UX, 초보 투자자 타겟 |
| **Webull** | 해외 | MTS + Web + Desktop | 고급 차트 도구, TradingView 통합 |
