# 이력서 오버레이 — 토스증권

## 소개 (Override)
디자인 시스템을 통해 개발 생산성을 극대화하는 프로덕트 디자이너 문호균입니다. Cursor AI로 14만 줄의 레거시 코드를 분석하여 패턴을 발견하고, 디자인 토큰 시스템과 컴포넌트 라이브러리를 구축하여 서버 구동 시간을 22초에서 0.2초로, 코드를 40% 감축한 경험이 있습니다.

비효율의 고리를 끊는 것에 집중합니다. 디자인 시스템 구축으로 5.5만 줄의 불필요한 코드를 제거하고, 3일 걸리던 엑셀 리포트를 1분으로 자동화하며, 듀얼뷰 UX 설계로 검색량 +26.43% 향상과 특허 등록을 달성했습니다. 패턴을 발견하고 원칙으로 정의하여 디자이너와 개발자 모두를 빠르게 만드는 시스템을 설계합니다.

## Strength (Override)
**Cursor AI 기반 레거시 코드 분석과 디자인 토큰 아키텍처 설계**
14만 줄의 레거시 코드에서 반복 패턴을 발견하고, 13,360줄의 SCSS를 디자인 토큰으로 전환하여 코드 라인 40% 감축에 기여했습니다. MUI 기반 컴포넌트 라이브러리를 제작하여 개발팀과의 협업 생산성을 향상시켰습니다.

**PC Web 환경에서의 복합 데이터 UX 설계**
PC Web 기반 듀얼뷰 싱크 시스템을 설계하여 여러 경로를 동시에 비교·분석할 수 있는 UX를 구현했습니다. 해당 기술의 독창성을 인정받아 특허 1건을 등록하였습니다.

**데이터 기반 문제 정의와 자동화 설계**
수동 데이터 다운로드 월 290회를 자동화하고, 리포트 작성 시간을 99% 단축하는 등 반복 작업의 병목을 데이터로 증명하고 시스템으로 해결한 경험이 있습니다.

## Application Motivation (Override)
토스증권의 TDS(Toss Design System)가 디자이너와 개발자를 1,000배 빠르게 만드는 비전에 깊이 공감합니다.

어센트코리아에서 14만 줄의 레거시 코드를 분석하여 반복 패턴을 발견하고, 디자인 토큰과 컴포넌트 라이브러리로 체계화하여 서버 구동 시간을 22초에서 0.2초로, 코드를 40% 감축한 경험이 있습니다. 이 과정에서 '패턴을 발견하고 원칙으로 정의하는' 디자인 시스템의 본질을 체득했습니다.

PC Web 환경에서 듀얼뷰 싱크 시스템을 설계하여 특허를 획득한 경험과, FE 4명과 긴밀히 협업하며 컴포넌트를 구조화한 경험을 바탕으로, 토스증권 PC Community 플랫폼의 비효율을 끊고 사용자와 동료 모두에게 빠른 경험을 제공하는 디자이너가 되겠습니다.
