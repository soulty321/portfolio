# 이력서 오버레이 — 당근마켓

## 소개 (Override)
B2B SaaS 프로덕트의 성과를 만드는 프로덕트 디자이너 문호균입니다. 사용자의 반복 작업 및 분석 패턴을 개선하여 키워드 등록률을 107% 증가시키고 검색량을 26.43% 증가시킨 경험이 있습니다.

핵심 기능의 필터 사용성을 개선하여 키워드 등록률 107% 증가시켰으며, 수동 데이터 다운로드 문제를 해결하여 월 290회의 불필요한 업무를 자동화했습니다. 또한, 커뮤니티 운영에서 챌린지를 통해 신규 참여율 4배 증가를 이끈 경험이 있습니다. 이익률 50%의 개인 브랜드를 운영하며 체득한 비즈니스 감각을 바탕으로, 데이터로 사용자의 불편함을 증명하고 논리적인 UX 설계로 비즈니스의 성장에 기여하였습니다.

## Strength (Override)
**퍼블리싱 코드 작성 경험과 Cursor를 통한 데이터 기반 디자인 시스템 설계 경험**
프로젝트의 복잡성을 나타내는 코드 라인 수의 40% 감소에 기여했습니다. Cursor AI를 통한 코드 내 패턴을 발견하고 13,360줄의 SCSS 코드를 정리하였습니다.

**신규 서비스 출시 및 기존 기능 개선**
데이터 분석 서비스 사용에 필요한 키워드 추적 기능과 경로 비교 기능을 설계하고, 사용자 리서치와 Beta 기능 사용성에 기반하여 출시한 사례가 있습니다.

**사용자 보상 경험 파악 및 시스템화**
운영 중인 자기계발 모임의 사용자들이 달성한 보상 경험을 확인하여 이를 신규 유저에게 적용하였고, 참여율이 4배 증가하며 지역 내 자기계발 모임 1위를 달성한 경험이 있습니다.

## Application Motivation (Override)
당근에서 '각자 할 거 하는 모임'을 직접 운영하며, 참여율 4배 성장·지역 1위라는 결과를 만들었습니다. 50대 이웃이 처음 모임에 나와 어색해하다가, 세 번째 참여에서 스스로 후기를 쓰는 순간을 옆에서 지켜보았습니다. '우리 엄마 아빠도 쓸 수 있을까?'라는 당근의 질문을 오프라인 현장에서 먼저 체득한 셈입니다.

저는 항상 '이 데이터가 사용자의 다음 행동을 바꾸는가?'를 먼저 묻습니다. B2B SaaS에서 키워드 등록률 107% 증가와 3일→1분 리포트 자동화를 이끌며, 사용자의 반복 행동 뒤에 숨은 진짜 문제를 데이터로 찾고 해결해왔습니다.

하이퍼로컬 현장에서 쌓은 사용자 감각과, SaaS에서 검증한 데이터 기반 문제 해결력이 만나는 지점이 당근의 비즈니스 프로필이라고 생각합니다. 입사 첫 달, 비즈니스 프로필을 쓰는 동네 사장님 10명을 직접 만나 첫 등록 과정을 관찰하고, 온보딩 개선 패턴을 제안하고 싶습니다.
