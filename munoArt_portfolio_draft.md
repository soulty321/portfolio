## 0. 메타 메시지 (포트폴리오 헤드라인)

> **무에서 유를 만드는 디자이너 — 음악가를 타겟으로 시장을 개척한 4년**

채용 평가자가 5분 안에 단 한 문장만 가져간다면, 위 한 줄이 남도록 모든 섹션이 설계되었습니다.

## 1. Overview

### 한 줄 정의
사업화 의도 0%로 시작한 5장의 일러스트가 1,008번의 인터랙션을 만들어내며 자발적 시장 신호로 변환된 후, 4년간 410건의 의뢰·굿즈·협찬으로 확장된 **1인 음악 굿즈 비즈니스**.

### 메타데이터
| 항목 | 값 |
|---|---|
| 기간 | 2018 - 2022 (4년) |
| 역할 | 작가 + 디자이너 + 리서처 + 가격 설계자 + 마케터 + 1인 운영자 |
| 트리거 | 2017.7.29 클래식에 미치다 게시물 (반응 782 / 댓글 151 / 공유 75) |
| 분류 | 개인사업 (1인) |

### 한 단락 요약 (Overview 본문)
**군악대 복무 중 관악기 키 메커니즘에서 발견한 "기계적 미감"**을 시각으로 번역한 5장의 일러스트가, "클래식에 미치다" 페이지에서 1,008번의 자발적 인터랙션을 만들어냈다. 댓글에 반복적으로 등장한 "내 악기 처음 보는 느낌"이라는 반응을 시장 신호로 해석하고, 군악대 내 1차 리서치를 통해 "악기 케이스 휴대"라는 페인포인트를 발견하여 네임택을 시작으로 굿즈 라인업을 확장했다. **4년간 410건의 제작·평균 이익률 50.52%·세계적 연주자 협찬 2건**을 운영하며, "디자인만 한 디자이너"가 아닌 "**고객의 문제를 발견·정의하고 수익까지 만들어낸 디자이너**"로 성장한 경험.

## 2. Problem

### 문제 정의 (한 줄)
**음악가는 자기 악기를 다르게 보고 싶어 했지만, 시장에는 그것을 진지하게 다룬 굿즈가 없었다.**

### 문제 발견의 데이터 근거 ▣

**1차 시장 신호 — 클래식에 미치다 게시물 (2017.7.29)**

| 지표 | 수치 |
|---:|---:|
| 총 반응 (Reactions) | **782** |
| 총 댓글 (Comments) | **151** |
| 총 공유 (Shares) | **75** |
| **총 인터랙션** | **1,008** |
| 부정 반응 | **0건** |
| 좋아요 비중 | 98.6% |

**댓글에서 직접 검증된 가설:**
- ★ "*난 내 악기 볼때마다 낯설다*" (사용자 효빈)
- ★ "*처음 보는 느낌*" (사용자 하다희)
- 누락 악기 요청 폭증: 오보에·색소폰·바이올린·호른·피아노·타악기 — 자발적 수요 표명

### 시장 공백의 정의
- 기존 음악 굿즈 시장은 **"귀여운·캐주얼한 표현"** 일색
- ★ "*악기의 자체를 세부적으로 표현하는 게 없었다*" — 진지·세부·프리미엄 스펙트럼의 부재
- 클래식 음악가가 자기 정체성을 표현할 시각적 수단의 빈약

### ★★ 사용자 재정의 (UX 사고의 핵심)
표면 사용자(연주자 본인)와 **실제 구매자(연주자의 팬·제자)** 의 분리를 발견:

> ★ "*유명 신예 연주자 분들이 직접 요청하기보단, 그분들의 제자나 팬들이 요청을 하였습니다. 그리고 실제로 그분들의 연주나 행사에 사용되는 것을 보면서 많은 뿌듯함을 느꼈죠.*"

→ 굿즈는 "음악가용"이 아니라 "**음악가에게 헌정·선물하기 위한**" 제품으로 사용자 정의를 재구성.

## 3. Strategy

### 4년간의 진화 로드맵

| 시점 | 사건 | 의사결정의 트리거 |
|---|---|---|
| 2017.7 | 클래식에 미치다 5장 업로드 | 사업화 의도 0% / 자기표현 / 동료의 SNS 공유에서 영감 |
| 2017–2018 | 댓글 모니터링 + 군악대 내 1차 리서치 | "굿즈 내달라" 댓글 → 음악가 휴대 행동 관찰 → "악기 케이스" 페인포인트 발견 |
| 2018 | 네임택 출시 | 악기 케이스에 부착 가능한 첫 굿즈로 결정 |
| 2018–2019 | 음악가 맞춤 일러스트 의뢰 | 팬·제자가 헌정용으로 의뢰 |
| **2019** | **피크 매출 328만원** | 최대 가동 시점 |
| 2019–2020 | 협찬 액티브 제안 | 본인이 한국 국적 세계적 연주자에게 직접 제안 → 2분 수락 |
| 2020 | 폰케이스·명함·비즈니스 브랜딩 확장 | "음악가도 명함을 주고받는다" / "폰케이스로 자기 악기 정체성 표현" 인사이트 |
| **2020** | **매출 99만원 (전년 1/3, 코로나 직격)** | 외부 환경의 객관적 임팩트 |
| 2021 | 매출 47만원 (지속 감소) | 음악가 활동 중단 → 굿즈 수요 부재 |
| 2022 | 사업 종료 → UX/UI 디자이너 전환 | "디지털 시대의 새 방향" 판단 |

### 비즈니스 모델 설계 (자산 분리 사고)

| 자산 분류 | 역할 | 가격 정책 |
|---|---|---|
| **일러스트** | 마케팅 자산 | 대부분 **무료 선물** — 작가 IP 노출 채널 |
| **굿즈** | 수익 자산 | **마진 40~50%** — 네임택 원가 7,800원 → 판매 ~10,000원대 |
| **협찬** | 레퍼런스 자산 | 무료 제공 — 노출·신뢰도 가치 회수 |

### Strategy 핵심 카피 ◇
> "**듣는 것**으로 시작했다 — 댓글, 군악대 동료, 단체 주문 학생들에게서 수요를 듣고, 그 수요를 가격·라인업·협찬으로 번역한 4년."

## 4. SolutionA — 두 가지 니즈를 하나의 솔루션으로

### 문제와 솔루션의 직접 연결

Problem에서 확인된 두 개의 니즈:

| 사용자 | 니즈 | 실제 반응 |
|---|---|---|
| **음악가 본인** | "자기 악기를 다르게 보고 싶다" | ★ "난 내 악기 볼 때마다 낯설다" / "처음 보는 느낌" |
| **팬·제자** | 음악가에게 헌정할 의미 있는 물건 | ★ "유명 신예 연주자 분들의 제자나 팬들이 요청을 하였습니다" |

---

### 니즈 ① — "처음 보는 내 악기"를 만드는 법

기존 시장이 외면한 것: 악기 자체를 진지하고 세부적으로 표현하는 것.

> ★ "*악기의 자체를 세부적으로 표현하는 게 없었다.*"

| 구성 요소 | 사용자에게 주는 것 |
|---|---|
| **Constant** — 기계적 미감, 메탈릭 재구축 | "처음 보는 내 악기" — 시장에 없던 시각언어로 낯섦을 실현 |
| **Variable** — 연주자 감성의 컬러·타이포그래피 | "나다운 내 악기" — 자신의 연주 스타일과 감성이 담긴 개인화 |

→ 두 요소의 결합이 "처음 보는 느낌"이라는 댓글 반응을 직접 만들어냈다.

> ★ "*악기 그림의 큰 스타일, 기계적인 느낌은 변하지 않아요. 하지만 컬러나 들어가는 타이포그래피 등은 그 사람의 연주 스타일이나 원하는 감성을 좀 넣으려고 하였습니다.*"

---

### 니즈 ② — 헌정의 의미를 담는 법 ▣

팬·제자의 헌정 니즈는 단순 주문이 아니다 — 상대방의 악기와 감성을 알아야 완성된다.

**부산예고 학생 단체 주문 사례:**
- 학생들이 직접 상담에 참여
- 제작 이유(우정·추억), 기재 정보, 원하는 감성을 직접 협의
- → 단순 굿즈가 아닌, "우리의 시간이 담긴 물건"이 되었다

---

### 결과: 개인화 경험이 브랜드 신뢰로 전환

> ★ "*수정 피드백은 없습니다. 저라는 작가가 좋아서 온 것이어서.*"

→ 개인화된 경험은 "이 작가라야만 한다"는 신뢰로 전환됐다.
→ 1인 비즈니스에서 브랜드 신뢰가 제품의 일부가 된 사례.

## 5. SolutionB — AS-IS / TO-BE

> 변화의 두 축: **규모(B) + 역할(D)** 결합

### AS-IS: 5장의 일러스트를 자기표현으로 그리는 작가
- 사업화 의도 **0%**
- 작품 수: **5장** (호른·플루트·클라리넷·트럼펫·트롬본)
- 동기: ★ "*내 작품을 세상에 펼쳐보고 싶다*"
- 역할: 작가 단일 역할
- 수익: 0원

### TO-BE: 리서치·가격·협찬·단체주문까지 운영하는 1인 비즈니스 운영자

| 변화 축 | AS-IS | TO-BE |
|---|---|---|
| **규모** | 5장 | **410건** (맞춤 굿즈 330 / 케이스 59 / 명함 12 / 브랜딩 9) |
| **역할** | 작가 1개 | 작가 + 리서처 + 가격 설계자 + 마케터 + 운영자 |
| **고객 정의** | 막연한 "관객" | 음악가 + 음악가의 팬·제자 (헌정용 구매) |
| **프로세스** | 영감 기반 단일 작업 | 댓글 분석 → 1차 리서치 → 가격 설계 → 단체 주문 그룹 인터뷰 |
| **레퍼런스** | 없음 | 한국 국적 **세계적 연주자 협찬 2건** + 부산예고 단체 |
| **수익** | 0원 | 누적매출 **6,652,400원** / 이익률 **50.52%** |

### SolutionB 핵심 카피 ◇
> "**작가에서 비즈니스 운영자로** — 사업화 의도 0%였던 5장이 시장의 신호를 듣고 410건의 운영으로 확장된 4년."

## 6. Impact

### 정량 임팩트 ▣

| 지표 | 수치 |
|---|---|
| 총 제작 건수 | **410건** |
| ↳ 음악가 맞춤 굿즈 | 330건 |
| ↳ 커스텀 케이스 | 59건 |
| ↳ 비즈니스 명함 | 12건 |
| ↳ 비즈니스 브랜딩 | 9건 |
| 누적 매출 (4년) | **6,652,400원** |
| 누적 원가 | 3,291,600원 |
| 누적 이익 | 3,360,800원 |
| **평균 이익률** | **50.52%** |

**연도별 매출 추이 (코로나 임팩트의 객관적 증거)**

| 연도 | 매출 | 비고 |
|---|---:|---|
| 2018 | 1,904,000원 | 사업화 첫 해 |
| 2019 | **3,284,000원** | 피크 |
| 2020 | 994,800원 | 코로나 직격 (전년 -70%) |
| 2021 | 469,600원 | 음악가 활동 중단 영향 지속 |

### 정성 임팩트 ★★★

**바이올리니스트 김덕우님 직접 DM (인용 그대로)**
> *"안녕하세요! 저는 바이올리니스트 김덕우 입니다! 너무 반갑습니다 ~~ ^^*
> *이번 공연 디자인 해주셔서 너무 감사합니다 !! 멋지게 만들어주신만큼 좋은 공연 하도록 하겠습니다! :)"*

→ 무노아트의 작품이 단순 굿즈가 아니라 **"공연 디자인"** 으로 인정된 결정적 증거.

**여정의 시작점과 정점**
- **트럼펫** — 군악대에서 처음 통찰을 떠올리게 한 시작점
- **바이올린** — 국내외 유명 연주자에게 작품이 전달되고 본인에게 직접 DM이 도착, "굉장히 큰 성취감"

### Lesson — 정직한 자기 회고
> ★ "*수익을 재투자했어야 한다고 생각합니다. 마케팅이라든지 새로운 기획이라든지 이런 부분에 투자를 했어야 되는데, 그 당시에는 이런 감각이 전혀 없었기 때문에 다소 수익의 재창출화가 많이 어려웠던 것 같아요.*"

→ 비즈니스 감각의 부재를 본인이 명확히 인지 → 다음 커리어(프로덕트 디자이너)에서 보완하는 영역으로 자연 연결.

### 종료 결정의 정당성
- **외부 환경**: 코로나로 음악가 활동 중단 → 굿즈 수요 부재 (매출 데이터로 입증)
- **산업 변화**: 디자인 업계의 디지털 전환
- **본인 결정**: "디지털 시대의 새 방향" 판단 → UX/UI 디자이너로 자연스러운 전환

### ★★★ 메타 검증 — 이 경험이 채용 결과를 만들었다
현 A사 본부장님 채용 시 인용:

> ★ "*디자이너가 앞으로는 디자인만 할 일은 크게 없다. AI가 발전해 가면서 더 그렇다고 느낀다. 디자인은 어떠한 문제를 정의하고 그것을 해결해 나가는 게 좀 더 중요한 롤이 돼가고 있다. 그래서 디자인만 한 디자이너가 아닌, **실제로 고객의 문제를 발견하고 정의하고 수익까지 만들어낸 경험**을 더 높게 샀다.*"

→ **무노아트는 사이드 프로젝트가 아니라, 채용 결과를 객관적으로 만든 입증된 경험.**

### Impact 마무리 카피 ◇

> ★ "*이때 얻어낸 어떠한 성취감이, 제가 디자이너로서 세상과 소통하게끔 만들어 줄 수 있었던 계기였다고 생각합니다.*"

> **무노아트는 작품 5장에서 시작해 4년 후 한 디자이너의 정체성이 된 프로젝트다.**
> **AI 시대의 디자이너가 갖춰야 할 "문제 발견 → 정의 → 수익화"의 엔드투엔드 사고를, 음악가를 타겟으로 시장을 개척하며 학습한 무에서 유를 만드는 디자이너의 첫 챕터.**

- [x] 뻔한 표현("직관적", "편리한") 사용 안 함
- [x] 기능 정의서화(단순 기능 나열) 안 함
- [x] 정량 데이터 2개 이상 (410건 / 50.52% / 1,008 인터랙션 등 다수)
- [x] 정성 데이터 인용 (김덕우 DM, 댓글 인용)
- [x] 사용자 중심 표현 (페르소나 최소화, 실제 댓글·DM 인용으로 대체)

### 영문 i18n 키 변환 시 권장 키 네이밍
- `munoArt_project_HEADLINE` — 무에서 유를 만드는 디자이너
- `munoArt_project_OVERVIEW_BODY`
- `munoArt_project_PROBLEM_DEFINITION`
- `munoArt_project_PROBLEM_DATA_INTERACTIONS` — 1,008
- `munoArt_project_STRATEGY_TIMELINE_*`
- `munoArt_project_SOL_A_PRINCIPLE` — Constant + Variable
- `munoArt_project_SOL_B_ASIS_LABEL`
- `munoArt_project_SOL_B_TOBE_LABEL`
- `munoArt_project_IMPACT_QUANT_*`
- `munoArt_project_IMPACT_QUOTE_KIM` — 김덕우 인용
- `munoArt_project_IMPACT_QUOTE_HEADCHIEF` — 본부장 인용
- `munoArt_project_IMPACT_CLOSING`

### 후속 작업 권장
1. 본 카피 검토 후 Figma `munoArt_project` 페이지에 반영
2. `key.json`에 한국어/영어 키 추가 (영문 번역은 별도 단계)
3. SVG 이미지 갱신 — 특히 SolutionB의 AS-IS / TO-BE 이미지를 본 문서의 새 정의("5장 작가 → 410건 운영자")에 맞게 재작성
