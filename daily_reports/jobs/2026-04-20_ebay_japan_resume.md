## Experience

### 어센트코리아 (Ascent Korea)
**프로덕트 디자이너** | 2023.03 - 현재

ListeningMind — B2B SEO SaaS 플랫폼의 핵심 기능 UX 개선 및 디자인 시스템 구축을 주도했습니다.

**[PathDual — 유저 여정 기반 비교 탐색 경험 설계]**
`기간: 2024.09 - 2024.11` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 기획 및 디자인 (듀얼뷰 UX 기획 주도, 특허 UI 설계, 퍼블리싱 성능 최적화)`
- 마케터의 분석 여정을 관찰하여 단편적 데이터 확인의 한계를 진단, '듀얼 뷰(Dual View)' UI를 설계
- 사용자가 두 맥락을 나란히 비교·분석할 수 있도록 고객 여정 전반의 사용자 경험을 체계적으로 재설계
- **결과:** 기능 내 검색량 +26.43% 증가, 관련 UI 특허 등록
- **방법론:** 사용자 인터뷰 → 여정 매핑 → 프로토타입 → A/B 검증

**[MyKeyword — 비즈니스 KPI + 유저 데이터 연결 설계]**
`기간: 2023.11 - 2024.07` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 디자인 (UX/UI 디자인 · 프로토타이핑 · QA)`
- 핵심 기능 사용률 저조(4.5%) 문제를 데이터로 진단, 유저 플로우와 인터랙션을 체계적으로 재설계
- 시계열 분석 기능 도입으로 반복적인 엑셀 다운로드 워크플로우를 대체
- **결과:** 월 290회 반복 과업 제거, 타깃 유저 일평균 사용량(DAU) 2배 증가, 키워드 등록 +107%(+4,752개)
- **방법론:** 행동 데이터 분석 → 문제 정의 → Hook Model 기반 인게이지먼트 설계

**[Design System — Figma-개발 간 체계적 핸드오프]**
`기간: 2025.01 - 2025.06` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 기획 및 디자인 (디자인 토큰 체계 설계, 컴포넌트 라이브러리 구축)`
- Cursor AI를 활용해 1.3만 줄의 중복 코드를 진단하고 23개 컴포넌트 디자인 시스템을 재구축
- Figma 토큰 체계·베리언트 설계로 FE 4명과의 핸드오프 효율 극대화
- **결과:** 서버 준비 시간 22.6초 → 0.2초(99% 단축), 디자인 부채 해결
- **협업:** PM 1명, FE 4명, BE 협업

**[MyKeywordFilter — 필터 UX 개선으로 숨겨진 시장 발굴]**
`기간: 2024.07.30 - 2024.11.08` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 기획 및 디자인 (문제 정의, 아이디에이션, QA, 데이터 분석)`
- 검색량 필터 기능의 UX 문제를 진단, 핵심 키워드 접근성을 저해하는 인터랙션 흐름을 재설계
- 유저 여정 내 필터 사용성 개선으로 기존에 접근하지 못하던 88% 시장 영역을 노출
- **결과:** 일평균 키워드 등록 0건 → 261건, 일 최대 3,700건 달성, 등록 정확도 향상(추가·삭제 횟수 12회 감소)
- **방법론:** 서비스 내 필터 사용 행동 분석 → 문제 정의 → 프로토타입 → QA

**[AI Agent — SEO 키워드 전략 AI 에이전트 설계]**
`기간: 2026.01 - 2026.02` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 디자인 (AI 사이드바 UX/UI 설계) · 프롬프트 엔지니어링 (프롬프트 고도화 주도)`
- 리서치 팀의 7단계 AI 파이프라인 구조를 직접 이해하고 AI 사이드바 UX/UI를 설계
- 최대 1,000개 키워드를 30~40초 만에 전략으로 전환하는 에이전트 플로우 및 프롬프트 고도화 주도
- **결과:** 출시 후 433명 사용자, 3,410회 분석 수행, 인당 평균 7.9회 재사용(높은 리텐션)

**[JourneyFinder — CDJ 자동 분석 솔루션 설계]**
`기간: 2025.09 - 2026.02` ｜ `직책: 프로덕트 디자이너` ｜ `직무: 프로덕트 디자인 (CDJ 시각화 및 데이터 테이블 UX/UI 설계 100%)`
- 연구소의 7단계 AI 파이프라인 구조를 직접 파악하여 각 단계의 데이터 분류 결과를 시각화
- 소비자 의사결정 여정(CDJ) 차트 및 데이터 수정 테이블 UX/UI 설계 100% 담당
- **결과:** 1~2일 걸리던 수작업 분석을 10분으로 단축, 2,050건+ 분석 프로젝트 성공적으로 지원
