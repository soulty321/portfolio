## 6. 강점-약점 대응 전략

### 강점 극대화
- **검색 여정 UX** → Qoo10의 상품 탐색/검색 여정에 직결되는 경험
- **특허 등록** → 독창적 UX 솔루션 증명
- **일본어 능력** → 현지 팀 커뮤니케이션 가능 (우대사항 직접 충족)
- **크로스펑셔널 협업** → PM/FE/BE 팀과의 실무 협업 경험 풍부

### 약점 중화
| 약점 | 보완 방법 |
|------|-----------|
| 경력 연차 부족 (5-10년 요구) | 프로젝트 임팩트 크기로 보완; "리드급 의사결정" 강조 |
| 이커머스 직접 경험 없음 | PathDual을 "검색 기반 커머스 여정"으로 프레이밍 |
| B2B → B2C 번역 필요 | "마케터의 분석 여정" → "소비자의 탐색 여정"으로 유사성 강조 |
| 모바일/B2C 경험 약함 | Dangen(당근 모임) 사이드 프로젝트로 모바일 감각 증명 |
| UI 비주얼 상대적 약점 | Cover/Overview 페이지 비주얼 퀄리티 집중 투자 |
