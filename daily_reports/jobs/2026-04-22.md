## 💡 지원 전략 팁

### 🚨 우선순위 1: 채널톡 (D-8, 4/30 마감)

**칵테일 전략 배합:**
- 문제해결 + 논리적 구조화: **45%** (B2B SaaS의 핵심 요구)
- 시스템 설계 + 개발 협업: **35%** (서버 22초→0.2초, 코드 40% 감축 케이스 어필)
- 비주얼 UI + 인터랙션: **20%** (보완 포인트지만 지나치게 강조 불필요)

**JD 미러링 용어:**
- `빠른 호흡의 문제해결` → 포트폴리오에 "프로토타입 사이클 단축" 사례 삽입
- `개발팀과 적극적인 커뮤니케이션` → React 퍼블리싱 경험, 개발 스펙 직접 협의 사례
- `비즈니스 제휴, 마케팅팀 협업` → PO/PM 겸직 경험, 크로스펑션 리딩 사례
- `관리자 및 고객용 제품` → B2B SaaS 이중 사용자(운영자 vs 최종 고객) UX 설계 경험

**강조할 포트폴리오 프로젝트:**
1. **디자인 시스템 구축** — 코드 40% 감축, 서버 최적화 수치 강조
2. **Hubble AI 에이전트** — 1,000개 키워드 → 40초 전략 인사이트, "빠른 호흡" 증명
3. **듀얼뷰 동기화 UI (특허)** — 문제 발견→해결→특허 등록 스토리라인

**채널톡 스테이지 분석:**
- 시리즈 C+ 단계, 글로벌 확장 중
- 원하는 디자이너상: "비즈니스 임팩트를 만드는 실행형 디자이너" — 속도와 품질 동시 요구
- Ride The Challenge: 연봉 30% 인상 제안 = 즉시 기여 가능한 경력자 선호
