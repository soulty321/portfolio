#### JD 키워드 분석 및 강조 포트폴리오 추천

| 보이저엑스가 원하는 것 | 매칭되는 문호균님의 경험 |
|---|---|
| AI 서비스 UX 설계 | Hubble — AI 에이전트로 키워드 전략 자동화 |
| 복잡한 기능을 직관적으로 | 영상 편집/스캔 툴 UX → 복잡계 단순화 경험 |
| 데이터 기반 개선 | 검색량 +26.4%, 키워드 등록 +4,752개 |
| 개발자와 긴밀한 협업 | React/퍼블리싱 역량 → 스펙 정밀 전달 |
| 스타트업 속도감 | PO+PD 겸직, 빠른 실행 경험 |

#### 칵테일 전략 배합 비율


> ⚠️ 비주얼 비중을 낮추되, "AI UX의 명확성"으로 포장하면 오히려 차별화 요소가 됩니다.

#### JD에서 미러링할 용어
