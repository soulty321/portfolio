**왜 최적인가?**

- 소규모 정예팀 운영 → PO/PM 겸 디자이너 경험 프리미엄
- 데이터/자동화 중심 제품 설계 역량 직접 활용 가능

**칵테일 전략 배합 비율**

| 축 | 배합 비율 | 포인트 |
|----|---------|--------|
| 문제해결·논리 구조화 | 45% | "1,000개 키워드 → 40초 인사이트" Hubble 사례, 검색량 +26.4% |
| AI/자동화 설계 | 30% | AI 에이전트 UX 설계, API 파이프라인 연결 UX |
| 시스템 설계 | 15% | 디자인 시스템 (서버 22초→0.2초, 코드 40% 감축) |
| 비주얼/UI | 10% | 최소한의 클린 UI, 특허 등록 듀얼뷰 동기화 |

**JD 미러링 용어 (보이저엑스)**
- "AI 워크플로우 UX" → "사용자가 AI 결과를 신뢰하고 조작할 수 있는 인터페이스"
- "프로덕트 임팩트" → 정량 지표 (처리 시간, 키워드 수, 코드 감축률)
- "소수 정예 팀 협업" → 1~10인 스타트업 PO/PM+디자인 겸업 경험

**추천 포트폴리오 프로젝트 순서**
1. **Hubble (AI 에이전트)** — AI 서비스 UX 설계 역량 직접 증명
2. **마이키워드 (ListeningMind)** — 데이터 기반 의사결정, +26.4% 검색량, +4,752 키워드
3. **디자인 시스템** — 시스템 사고력, 개발 협업, 정량 성과
